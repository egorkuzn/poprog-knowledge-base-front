# Подробный тест по качеству сайта (черновик)

> Документ-заготовка. Будет расширяться по мере уточнения методики оценки UX/UI.

## Цель

Собрать детальную обратную связь о качестве пользовательского интерфейса и удобстве сценариев портала.

## Инструкция

1. Пройдите разделы теста последовательно.
2. Для каждого пункта поставьте оценку по шкале от 1 до 5.
3. Добавьте комментарии к пунктам с оценкой ниже 4.

## Раздел A. Навигация

- Понятность структуры меню.
- Скорость нахождения нужного раздела.
- Удобство работы в desktop/compact/mobile режимах.

## Раздел B. Поиск

- Понятность строки поиска.
- Релевантность результатов.
- Понятность переходов на внешние ресурсы.

## Раздел C. Контент

- Читаемость текстов.
- Понятность карточек публикаций/кейсов.
- Удобство просмотра на мобильных устройствах.

## Раздел D. Чат с ИИ

- Понятность сценария отправки сообщений.
- Скорость ответа интерфейса.
- Удобство длинного диалога.
